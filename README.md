# Code Lock Puzzle

An Android puzzle game made with Jetpack Compose where the player must convert the starting number into the target number using the given operations within limited moves.[web:9][web:7]

## Gameplay

Each round gives you:
- A random **start** number
- A random **target** number
- A total of **10 moves**

Available operations:
- `+1`
- `+3`
- `x2`
- `reverse`
- `sum`
- `reset`

The challenge is to find the correct sequence of operations before the moves reach zero.
